Hey Arve. I am not sure if Boxer works for 10.6.8.

Apparently there is some kind of networking support. You can read more about this in the FAQ of Boxer.

http://boxerapp.com/guide/adva...

As this networking works over IP I guess you could relatively simply play in a local LAN on 2 different Macs.

On the internet the one of you both who acts as host for the session has to forwards ports: 5479 and 5482(TCP) on your router to the local IP of your network. Your friend then can enter your public facing ip.

In order to find out this IP: http://www.whatismyip.com/

Here is a guide how you forward ports on a router: http://portforward.com/english...

I recently installed "Die Siedler II" on Mac OS X - this is how I pulled
it of.

  • If you don't have the game - go buy it
  • Open WineBottler
  • Create custom prefix
  • Select the .exe file and check one of the DirectX dll's

  • Install it
  • Now right click on the program WineBottler created
  • Open the package
  • Copy the Settlers II folder from Contents/Resources/drive_c/COG
    GAMES/ into another Folder
  • Download BoxerApp

  • Click onto "Import new Game"
  • Drag the Settlers II folder onto the window

  • Thats it! Launch the game!

Well. That's it! Enjoy.
